# Linux

Linux API bindings for [Rux](https://rux-lang.dev). All functions follow the official Linux Kernel API.

## Installation

You can install the package using Rux CLI

```sh
rux add Linux
```

Add `Linux` to your `Rux.toml`:

```toml
[Dependencies]
Linux = "*"
```
